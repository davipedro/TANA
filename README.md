# ⛩ TANA - Sistema de Reservas para Restaurantes

TANA (“tá na mesa”) é uma aplicação voltada para o gerenciamento inteligente de reservas em restaurantes.

O sistema permite que diferentes restaurantes administrem suas mesas, horários e reservas de forma centralizada, enquanto oferece aos clientes uma experiência fluida para buscar estabelecimentos, realizar e acompanhar suas reservas em tempo real.
